# Phase 0: WanderApp – Vision & Validierung  
**Status**: ✅ Abgeschlossen (Stand: 2025-04)  
**Ziel**: MVP mit fokussierter Zielgruppe, bevor Skalierung beginnt  

## 🎯 One-Sentence-USP  
> *Die einzige Wander-App, die **ohne Login, ohne Komplexität und zuverlässig offline** funktioniert – speziell auf Barrierefreiheit und einfache Steuerung für Senioren (65+) optimiert.*

## 🔑 Kerninsight  
Senioren nutzen bereits Smartphones, scheitern aber an:  
- komplexen Einstellungen  
- fehlendem Offline-Vertrauen  
- zu kleiner Schrift / unklaren Icons  

→ **Wir ersetzen Komplexität durch Vertrauen.**

## 🚀 MVP-Fokus (nicht im MVP)
❌ Social sharing  
❌ Klimadaten  
❌ AI-generierte Touren  
✅ *Barrierefreie Navigation mit Sprache & Grobfilterung*

## 📊 Erfolgskriterien für Phase 1
| Metrik | Zielwert |
|--------|----------|
| Installationsrate (ohne Login) | >80 % innerhalb von 30 Sekunden |
| Offline-Funktion getestet (Wandern im Wald) | 95 % Nutzer:innen schließenTour erfolgreich ab |
| Barrierefreiheit (WCAG 2.1 AA) | Ja – Schrift ≥18pt, Sprachausgabe, Tasten ≥48dp |

## 🧠 Quellen
- 12 Interviews mit Senioren (Mai 2025, Region Südbaden)  
- Komoot/AllTrails Usability-Studien (2023–2024)  
- WCAG 2.1 Guidelines (W3C)
